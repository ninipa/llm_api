# Public Samples

此目录用于放置可以公开保留在 GitHub 上的最小示例文件。

设计目标：

- 不包含私有素材
- 不包含实验结果
- 不依赖本机路径
- 只用于说明 `ask_text_file()` 和 CLI 文件输入的基本用法

当前文件：

- `demo_input.json`
  - 最小 JSON 输入示例
- `demo_script.command`
  - 最小 shell 脚本示例

当前没有内置图片示例。

原因：

- 正式支持的图片格式是二进制文件
- public 仓库当前只保留最小文本类示例
- 图片调用请改用你自己的 `.png/.jpg/.webp/...` 文件
